📚 Introdução

O MCP Sankhya (Model Context Protocol) é um conjunto de ferramentas que permite interagir com o banco de dados Sankhya ERP de forma segura e eficiente. Este serviço oferece duas categorias principais de ferramentas:

🔍 Ferramentas SQL

Execute queries SELECT, explore tabelas, schemas e relacionamentos do banco de dados Sankhya.

Apenas Leitura

📖 Dicionário de Dados

Acesse metadados completos das tabelas, campos, opções e relacionamentos do sistema Sankhya.

Metadados

🔐 Autenticação

Todas as requisições requerem autenticação JWT. Para obter um token, faça login com suas credenciais Sankhya:

Obter Token de Acesso

POST /auth/login

Corpo da Requisição

{
  "username": "SEU_USUARIO",
  "password": "SUA_SENHA"
}

Resposta

{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "refreshToken": "eyJhbGciOiJIUzI1NiIs...",
  "tokenType": "Bearer",
  "expiresIn": 3599
}

Usando o Token

Adicione o token no header de autorização:

Authorization: Bearer SEU_TOKEN_AQUI
⚠️ Segurança: Nunca exponha seus tokens ou credenciais em código público. O token expira em 1 hora.

🔍 Ferramentas SQL

Ferramentas para executar queries e explorar a estrutura do banco de dados.

Executar Query SQL

POST /mcp/query 
{
  "query": "SELECT TOP 10 CODPARC, NOMEPARC FROM TGFPAR WHERE CODTIPPAR = ?",
  "params": [1],
  "limit": 100
}

Listar Tabelas

GET /mcp/tables?limit=100

Obter Schema de Tabela

GET /mcp/tables/:tableName/schema

Buscar Tabelas

GET /mcp/tables/search?term=PARCEIRO

Buscar Campos

GET /mcp/fields/search?term=CODPARC

Chaves Primárias

GET /mcp/tables/:tableName/primary-keys

Relacionamentos

GET /mcp/tables/:tableName/relationships

📖 Dicionário de Dados

Acesse informações detalhadas sobre tabelas, campos e relacionamentos do sistema Sankhya.

Listar Tabelas do Dicionário

GET /mcp/dictionary/tables?limit=100

Detalhes de uma Tabela

GET /mcp/dictionary/tables/:tableName

Listar Campos de uma Tabela

GET /mcp/dictionary/tables/:tableName/fields

Detalhes de um Campo

GET /mcp/dictionary/tables/:tableName/fields/:fieldName

Opções de um Campo (Enumerados)

GET /mcp/dictionary/tables/:tableName/fields/:fieldName/options

Propriedades de Interface

GET /mcp/dictionary/tables/:tableName/fields/:fieldName/properties

Instâncias de uma Tabela

GET /mcp/dictionary/tables/:tableName/instances

Relacionamentos no Dicionário

GET /mcp/dictionary/tables/:tableName/relationships

Buscar Tabelas

GET /mcp/dictionary/tables/search?term=PARCEIRO

Buscar Campos

GET /mcp/dictionary/fields/search?term=CODPARC

💻 Exemplos de Uso

1. cURL - Listar Tabelas do Dicionário

TOKEN="SEU_TOKEN_AQUI"

curl -X GET "http://localhost:3027/mcp/dictionary/tables?limit=10" \
  -H "Authorization: Bearer $TOKEN" \
  -H "x-database: TREINA"

2. cURL - Buscar Parceiros

TOKEN="SEU_TOKEN_AQUI"

curl -X POST "http://localhost:3027/mcp/query" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -H "x-database: TREINA" \
  -d '{
    "query": "SELECT TOP 10 CODPARC, NOMEPARC FROM TGFPAR WHERE NOMEPARC LIKE ?",
    "params": ["%ACME%"]
  }'

3. cURL - Verificar Estrutura de Tabela

TOKEN="SEU_TOKEN_AQUI"

curl -X GET "http://localhost:3027/mcp/tables/TGFPAR/schema" \
  -H "Authorization: Bearer $TOKEN" \
  -H "x-database: TREINA"

4. JavaScript (Node.js)

const axios = require('axios');

const BASE_URL = 'http://localhost:3027';
const TOKEN = 'SEU_TOKEN_AQUI';

const client = axios.create({
  baseURL: BASE_URL,
  headers: {
    'Authorization': `Bearer ${TOKEN}`,
    'Content-Type': 'application/json'
  }
});

async function listarTabelas() {
  const response = await client.get('/mcp/dictionary/tables', {
    params: { limit: 10 },
    headers: { 'x-database': 'TREINA' }
  });
  return response.data;
}

async function executarQuery(query, params = []) {
  const response = await client.post('/mcp/query', {
    query,
    params
  }, {
    headers: { 'x-database': 'TREINA' }
  });
  return response.data;
}

// Exemplo de uso
(async () => {
  const tabelas = await listarTabelas();
  console.log('Tabelas:', tabelas.data);
})();

5. Python

import requests

BASE_URL = 'http://localhost:3027'
TOKEN = 'SEU_TOKEN_AQUI'

headers = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json'
}

def listar_tabelas(limit=10):
    response = requests.get(
        f'{BASE_URL}/mcp/dictionary/tables',
        params={'limit': limit},
        headers={**headers, 'x-database': 'TREINA'}
    )
    return response.json()

def executar_query(query, params=None):
    response = requests.post(
        f'{BASE_URL}/mcp/query',
        json={'query': query, 'params': params or []},
        headers={**headers, 'x-database': 'TREINA'}
    )
    return response.json()

# Exemplo de uso
if __name__ == '__main__':
    result = executar_query(
        'SELECT TOP 5 CODPARC, NOMEPARC FROM TGFPAR'
    )
    print(result)

6. Selecionar Banco de Dados

Use o header x-database para escolher qual banco acessar:

Header Descrição
x-database: TREINA Banco de treinamento (padrão, seguro)
x-database: TESTE Banco de testes
x-database: PROD Banco de produção

🛠️ Ferramentas MCP Disponíveis

O servidor MCP oferece as seguintes ferramentas que podem ser usadas com clientes MCP:

Ferramentas SQL

query_sql_server

Executa queries SQL SELECT no SQL Server Sankhya. Retorna até 10000 registros.

{
  "query": "SELECT TOP 10 * FROM TGFPAR WHERE CODTIPPAR = ?",
  "params": [1],
  "limit": 100
}

list_tables

Lista todas as tabelas disponíveis que o usuário tem permissão.

get_table_schema

Retorna a estrutura/colunas de uma tabela específica.

{"tableName": "TGFPAR"}

search_tables

Busca tabelas por nome ou descrição no dicionário.

{"term": "PARCEIRO", "limit": 100}

search_fields

Busca campos por nome ou descrição em todas as tabelas.

{"term": "CODPARC", "limit": 100}

get_table_primary_keys

Retorna as chaves primárias de uma tabela.

{"tableName": "TGFPAR"}

get_table_relationships

Retorna os relacionamentos (foreign keys) de uma tabela.

{"tableName": "TGFPAR"}

Ferramentas do Dicionário de Dados

dictionary_list_tables

Lista todas as tabelas do dicionário de dados Sankhya.

{"limit": 100, "offset": 0}

dictionary_get_table_details

Obtém informações detalhadas de uma tabela específica.

{"tableName": "TGFPAR"}

dictionary_list_fields

Lista todos os campos de uma tabela do dicionário.

{"tableName": "TGFPAR", "limit": 500}

dictionary_get_field_details

Obtém detalhes completos de um campo específico.

{"tableName": "TGFPAR", "fieldName": "CODPARC"}

dictionary_get_field_options

Lista todas as opções/valores possíveis de um campo enumerado.

{"tableName": "TGFPAR", "fieldName": "ATIVO"}

dictionary_get_field_properties

Retorna propriedades de interface de um campo.

{"tableName": "TGFPAR", "fieldName": "CODPARC"}

dictionary_get_table_instances

Lista as instâncias (agrupamentos lógicos) de uma tabela.

{"tableName": "TGFPAR"}

dictionary_get_table_relationships

Retorna os relacionamentos de uma tabela no dicionário.

{"tableName": "TGFPAR"}

dictionary_search_tables

Busca tabelas no dicionário por nome ou descrição.

{"term": "PARCEIRO", "limit": 100}

dictionary_search_fields

Busca campos por nome ou descrição em todas as tabelas.

{"term": "CODPARC", "limit": 100}

⚠️ Limitações e Segurança

🔒 Restrições de Segurança:
  • Apenas queries SELECT são permitidas
  • Todas as operações são somente leitura
  • Rate limiting ativo (60 requests/minuto)
  • Limite de 10000 registros por query
  • Acesso baseado em permissões do usuário Sankhya
